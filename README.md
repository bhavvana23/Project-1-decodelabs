Cleaned E-Commerce Orders Dataset | DecodeLabs Project 1 

This repository contains a cleaned and structured e-commerce orders dataset prepared as part of the DecodeLabs Phase 1 Data Analytics Project. The dataset was processed to improve data quality, consistency, and usability for business analysis and reporting.

📂 Dataset Information

- File Name: "Cleaned_Dataset_DecodeLabs_P1.xlsx"
- Worksheet: "Cleaned Data"
- Project: DecodeLabs Project1 – Data Cleaning & Preparation

📊 Dataset Features

The dataset includes key business-related information such as:

- Order ID
- Order Date
- Customer ID
- Product Category
- Quantity Purchased
- Unit Price
- Total Order Value
- Payment Method
- Order Status
- Shipping Address
- Tracking Number
- Items in Cart
- Coupon Code
- Referral Source

🛠 Data Cleaning Performed

- Removed duplicate records
- Handled missing and inconsistent values
- Standardized column names and data formats
- Corrected data inconsistencies
- Verified calculated fields such as Total Price
- Prepared the dataset for analysis and visualization

📈 Possible Business Insights

- Product-wise sales performance
- Revenue and order trend analysis
- Customer purchasing behavior
- Payment method distribution
- Order status and return analysis
- Coupon usage effectiveness
- Referral source performance
- Business reporting and dashboard creation

🎯 Project Objective

The objective of this project was to transform raw e-commerce order data into a clean, reliable, and analysis-ready dataset that can support data-driven business decisions and reporting.
